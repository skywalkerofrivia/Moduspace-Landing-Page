A finished LEGO build is a structure, not a figure. It has a footprint, a centerpiece, and almost always a long horizontal axis you didn't plan around. Sizing a case is mostly about respecting that footprint.

Plan around the footprint, not the height

The Bonsai Tree (10281) has a 22×17cm footprint. The Millennium Falcon UCS (75192): 84×56cm. The single most common mistake is sizing for height — LEGO models are wider and longer than they are tall.

Sizes for the sets we see most

  • Bonsai / Orchid / Botanical: 30×30×30cm interior
  • Modular Buildings: 35×35×40cm
  • Architecture skylines: 60×15×25cm
  • UCS Falcon / Star Destroyer: 100×60×40cm
  • Hogwarts Castle: 70×45×60cm

The next set is already coming

If you've bought one Bonsai, you'll buy three. Plan a horizontal run of identical cases now, even if you only fill the first one this year. Matching cases reads as a collection. Mismatched ones read as clutter.
